# 통상임금 계산기

간단한 한국 통상임금 계산기입니다. 기본급과 통상적 수당을 더해 월/일/시 통상임금을 계산하며,
선택적으로 초과근무 수당도 계산할 수 있습니다.

## 사용 방법

1. `tong_sang_ingeum_calculator.py` 파일을 실행합니다.
2. 기본급, 통상적 수당, 월 유급 기준 일수, 하루 근무시간을 입력합니다.
3. 초과근무 시간이 있으면 입력하고 가산율을 입력하면 초과근무 수당을 계산합니다.

## 실행 예

```bash
python tong_sang_ingeum_calculator.py
```

## 기본값

- 월 유급 기준 일수: 30
- 하루 근무시간: 8
- 초과근무 가산율: 1.5
